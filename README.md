````md
# 🦸 MY HERO ACADEMIA QUIZ GAME ⚡
## Boku no Hero Academia Ultimate Quiz

Um jogo de quiz completo, moderno e interativo inspirado no universo de **My Hero Academia**.

O projeto foi desenvolvido com foco em:
- Visual anime/shounen
- Gameplay dinâmica
- Alta rejogabilidade
- Sistema avançado de pontuação
- Interface responsiva
- Experiência cinematográfica

# 🏆 Sistema de Pontuação

## Sistema Inteligente
O jogo possui:

- Pontuação dinâmica
- Combo de acertos
- Multiplicador por dificuldade
- Sistema de ranking
- Anti-repetição de perguntas

## Rankings Finais

Dependendo da pontuação, o jogador recebe um título:

| Pontuação | Rank |
|---|---|
| 0% - 20% | Estudante da U.A. |
| 21% - 40% | Herói em Treinamento |
| 41% - 60% | Pro Hero |
| 61% - 80% | Símbolo da Paz |
| 81% - 95% | Nível All Might |
| 96% - 100% | One For All Supremo |
````

```javascript
{

    "Toshinori Yagi",
    "Izuku Midoriya",
    "Shota Aizawa",
    "Mirio Togata"
  ],
```

Instale as dependências:

```bash
npm install
```

Execute:

```bash
npm run dev
```

Build de produção:

```bash
npm run build
```
